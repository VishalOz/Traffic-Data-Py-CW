This coursework assignment for our first-year, first-semester programming module focuses on developing fundamental file handling skills in Python. The project involves:
Reading and writing files – working with structured CSV files containing traffic data.
Extracting and processing data – parsing survey results and organizing them into meaningful structures.
Implementing robust input validation systems – ensuring correct date formats, handling leap years, and restricting input ranges.
Data visualization with Tkinter – presenting traffic survey results in the form of a histogram, showing vehicle frequency across different hours of the day.
GUI design – constructing an interactive, user-friendly interface with legends, labels, and scalable bar charts.
Multi-file support – enabling the selection and analysis of multiple survey datasets in a single run.
This assignment helped us combine theoretical programming concepts (functions, loops, conditionals, error handling, and dictionaries) with practical applications (GUI development, file handling, and data visualization).
